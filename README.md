Installation
```
$ git clone https://cpvent@bitbucket.org/intouchretail/store-hero-client.git
```
```
$ cd store-hero-client
```
```
$ npm install
```
```
$ gulp install
```
```
$ ionic platform add ios
```
Web Build
```
$ gulp build
```
IOS Build
```
$ ionic build
```
IOS Emulate
```
$ cordova run ios --target="iPad"
```

see Cordova for documentation.
