Sails | Angular | Ionic

Installation

```
$ git clone https://github.com/carlospliego/saijs.git
```

```
$ cd saijs
```

```
$ npm install
```

```
$ bower install
```

Mobile Platforms

```
$ ionic platform add ios
```

IOS Build & Emulate

```
$ ionic build-ios
```
